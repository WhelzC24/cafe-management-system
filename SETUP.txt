====================================================
  COZY CORNER CAFÉ — Setup Guide
====================================================

REQUIREMENTS
------------
- PHP 7.4+ with mysqli and fileinfo extensions
- MySQL 5.7+ or MariaDB 10.3+
- Web server: Apache or Nginx
- Writable uploads/ folder (chmod 755)

----------------------------------------------------
QUICK START
----------------------------------------------------

1. PLACE FILES
   Copy all files into your web root, e.g.:
     /htdocs/activity/          (XAMPP)
     /var/www/html/activity/    (Linux Apache)

   Folder structure must stay:
     activity/
       cafe/           ← public customer website
       img/            ← chatbot GIF
       uploads/        ← product image uploads (must be writable)
       db.php
       *.php           ← staff/admin backend
       style.css       ← admin/staff styles

2. IMPORT DATABASE
   Run in MySQL:
     mysql -u root -p < database.sql

   Or in phpMyAdmin: Import → choose database.sql

   This creates the 'web_system' database with all
   tables, the admin account, and 16 sample products.

3. CONFIGURE DATABASE (if needed)
   Edit db.php OR set environment variables:
     DB_HOST = localhost
     DB_USER = root
     DB_PASS = (your password)
     DB_NAME = web_system
     DB_PORT = 3306

4. SET UPLOADS PERMISSIONS
   chmod 755 uploads/
   (Apache/Nginx must be able to write to this folder)

5. VISIT THE SITE
   Customer site:  http://localhost/activity/cafe/index.html
   Staff login:    http://localhost/activity/login.html
